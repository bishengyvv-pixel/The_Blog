# 1

这是我的个人技术博客，用于沉淀技术知识、记录学习历程。

## 技术栈

- **React** - 前端框架
- **Vite** - 构建工具
- **TypeScript** - 类型安全
- **pretext** - 文本排版引擎

## 功能特性

1. 支持分类和标签组织
2. 专题系列文章
3. 时间线归档
4. 深浅色主题切换
5. 响应式设计
6. 代码高亮（VS Code 风格）
